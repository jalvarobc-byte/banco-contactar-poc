# Banco Contactar PoC

Landing page para la prueba de concepto de Banco Contactar con Amazon Connect Chat.

## Archivos
- `index.html`: página principal con el widget de Amazon Connect integrado.
- `css/style.css`: estilos visuales.
- `js/script.js`: comportamiento básico del botón principal.

## Publicación en GitHub Pages
1. Crear un repositorio llamado `banco-contactar-poc`.
2. Subir estos archivos.
3. Ir a Settings > Pages.
4. En Source seleccionar `Deploy from a branch`.
5. Seleccionar branch `main` y carpeta `/root`.
6. Guardar.
7. Abrir la URL publicada.
